# Clove Dental Landing Page

This project is a responsive landing page built using **HTML, CSS, and Vanilla JavaScript** based on the provided Figma design.

## Live Demo

https://unbundle-assignment.vercel.app

## GitHub Repository

https://github.com/sdev48/Unbundle-Assignment

## Tech Stack

* HTML
* CSS
* Vanilla JavaScript

## Features

* Fully responsive design (desktop, tablet, mobile)
* Clean and semantic HTML structure
* Modular section loading using JavaScript
* Horizontal card sliders

## Sections Implemented

* Navbar
* Hero Section
* Treatment Section
* Transformation Section
* Testimonials
* Locations
* Footer

## Deployment

The project is deployed using **Vercel**.
